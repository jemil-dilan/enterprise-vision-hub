# Project Memory

## Core
Supabase for backend, DB, Storage, and Auth (Admin, Client, Employee).
Custom context-based i18n (FR/EN) using localStorage 'dc_lang'. No external i18n libs.
Typography: Syne for headings, Inter for body.
Brand Palette: Bleu Nuit #0F172A (primary), Vert D&C #1D9E75 (secondary), Or #D4A017 (accent), bg #F8FAFC.
Premium UI: fluid transitions, parallax glows, scroll-triggered animations, floating WhatsApp FAB.
Public navbar limited to: Accueil, Services, À Propos (ancre), Contact. E-commerce routes exist but hidden from menu.
Shipchandler products strictly excluded from public catalog view.
People imagery: predominantly Black, occasionally Chinese or White. Founder portrait is exempt.
